# Data Directory

This folder is reserved for small sample images and camera intrinsics that can be used with the demo apps.

## Usage

- Place your own image pairs in `data/sample/`.
- Use a small image pair and a YAML camera intrinsics file.
- Do not commit large datasets to the repository.

## Suggested datasets

- KITTI Odometry
- EuRoC MAV
- TUM RGB-D
- ETH3D

The first version of this project is designed to run on two user-provided images and a matching intrinsics file.
