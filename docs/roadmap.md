# Roadmap

This project is organized to grow from a clean two-view SfM baseline to a more complete visual SLAM system.

## Short-term goals

- Complete two-view reconstruction demo
- Add bundle adjustment with Ceres
- Support sparse point cloud export and basic processing
- Add monocular visual odometry skeleton

## Medium-term goals

- Improve pose estimation robustness
- Add local bundle adjustment and keyframe management
- Add point cloud filtering and registration utilities
- Support common datasets and camera models

## Long-term goals

- Loop closure and pose graph optimization
- Visual-inertial odometry
- Real-time performance for XR applications
- Mobile / embedded deployment
